# SSH Connection

## Windows

- Install [putty](http://www.putty.org)
- Install [SuperPutty](https://github.com/jimradford/superputty)
- Configure SuperPutty

## Mac OS

- Use native Terminal app

## To connect to a remote device via SSH

```bash
ssh user@ip.add.re.ss
````

> Example

```bash
ssh pi@192.168.1.22
```
