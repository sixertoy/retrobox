# Overclocking

## Via Raspi Config

```bash
cd ~
sudo raspi-config
» Select Overclocking
» Select RPi2
sudo reboot
```

## Via Config.txt

> **RPi2 Overclocking**

```bash
cd ~
sudo /boot/config.txt

sudo reboot
```

> **Default values**

```bash
cd ~
sudo /boot/config.txt

sudo reboot
```

## Disable Overclocking

> **Doesnt' work with NOOBS installed**<br>
> Hold shift key at boot
