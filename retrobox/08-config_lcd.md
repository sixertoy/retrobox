# 8. Overclocking & Config LCD

> See sudo nano /etc/modules for FPS and Frequency

```bash
sudo nano /boot/config.txt
gpu_mem=512

hdmi_drive=2
hdmi_force_hotplug=1
hdmi_group=2
hdmi_mode=87
hdmi_cvt=320 240 60 1 0 0 0
sdtv_mode=2 pal
```
